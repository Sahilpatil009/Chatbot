# 🤖 Chatbot

A simple **AI-powered chatbot application** that interacts with users and provides automated responses. The project demonstrates how conversational systems work using **Python and natural language processing concepts**.

This project is useful for learning **chatbot architecture, text processing, and conversational AI basics**.

---

# 🚀 Features

* 💬 Interactive chatbot interface
* 🧠 Basic Natural Language Processing (NLP)
* ⚡ Fast response to user queries
* 📚 Predefined responses for common questions
* 🖥️ Simple and easy-to-understand project structure

---

# 🛠️ Tech Stack

**Programming Language**

# ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Sahilpatil009/Chatbot.git
```

### 2️⃣ Navigate to project folder

```bash
cd Chatbot
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Running the Chatbot

Run the following command:

```bash
python chatbot.py
```

Then start chatting with the bot in the terminal.

Example:

```
You: Hello
Bot: Hi! How can I help you today?

You: What is AI?
Bot: AI stands for Artificial Intelligence.
```
